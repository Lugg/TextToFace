Anleitung

Für die Software "TextToFace"

Mithilfe der Text to Face Software wird anhand des eingegebenen Textes ein Phantombild und Steckbrief erstellt. Das Ergebnis beschreibt den Autor des Textes genauer. Diese Software hilft Ihnen bei der Idenifikation von Personen anhand eines Textes. Sollten Sie sich schon mit der Software auskennen, können Sie über den "Text to Face" Button direkt zur Software gelangen. Viel Spaß und Erfolg bei der Verdächtigtensuche.

1. Schritt:

Sie geben einen Text in das Textfeld ein. Der Text sollte eine Mindestlänge von xx Zeichen einhalten, damit eine aussagekräftige Analyse möglich ist.

2. Schritt:

Sie schicken uns Ihren Text, indem Sie den TextToFace Button drücken.

3. Schritt:

Bitte warten Sie, bis die textuelle Psychoanalyse fertig ist. Sie werden daraufhin automatisch zum Phantombild weitergeleitet.

(optional)4. Schritt:

Zum Herunterladen des Phantombildes das Bild per Rechtsklick speichern.
